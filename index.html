注意事項
  • 本ツールでの索引整形は一定の法則にしたがって機械的に一括処理しています。
  • 例外的なデータには対応できません。精度にも限界がありますので必ず目でも校正してください。
  • 本ツールが外部にデータを送信することは一切ありません。

ダウンロード方法（初回のみ）

リリースページの一番上に表示されている exe ファイルをクリックするとダウンロードが始まります。100MB を超える巨大なファイルなので、余裕のある回線をご利用ください。

ダウンロード後の流れ

個人開発のアプリなのでダウンロード時に警告が表示されます（下記は Microsoft Edge の例）。信頼していただければ、下記の手順で有効化してください。

  1. マウスを乗せると表示されるメニューから「保存」を選択。
  2. あらためて警告が出るので「詳細表示」をクリック。
  3. 「保持する」をクリックするとダウンロードが正式に始まります。
  4. ダウンロード先のフォルダを開くと yomi.exe というファイルがダウンロードされているはず。
  5. アプリはこの exe 単体で動作します。適当なフォルダを作って移動させましょう。

初回起動時

おそらく初回の起動時に、Windows からの警告が表示されます。同じく信頼していただけましたら、下記の手順で有効化してください。

  1. 「PC が保護されました」という表示の場合は「詳細情報」をクリック。
  2. 表示される「実行」ボタンをクリックするとアプリが起動します。

作業手順

1. 名寄せした索引項目の読み情報を取得する

名寄せが済んだ索引をテキストボックスに貼り付けてGO!のボタンをクリックすると読み情報が取得されます。

  • 結果は「元の文字」「読みがな」「解析の詳細」の3列です。
  • オプションの意味：
    • Skip inside: () / []：括弧内の文字（見よ元項目の情報など）を無視します。
    • Skip nombles or reference (for book index)：ノンブル情報（全角空白2個の後ろにある数字）、見よ先情報（　→の後ろにある文字）を無視します。

2. 結果を Microsoft Excel に貼り付けてチェックする

COPY!ボタンで変換結果をコピーします。Excel を開いて貼り付けると楽にチェックできます。

  • 一番右の列で、どの単語をどう読んでいるのかがわかります。
  • この先使用するのは左の2列だけです。3列目を見ながら、変換にミスがあれば2列目を修正していきます。

3. 必要な列をコピーする

読みがなのチェック・修正が終わったら、左2列をコピーして以降のステップ（並べ替え）に進んでください。

  • 配列読みは次の並べ替えのステップで自動生成します。

並べ替え

  • 名寄せした索引項目と読みの2列を貼り付けてください。
  • 内部で読みの情報を配列読みに変換して並べ替えます。
    • 最優先は配列読みの情報。
    • 配列読みが同じ場合は読みで昇順ソート。
    • 読みも同じ場合は項目の文字コード昇順（同じ文字種がまとまります）。
  • 1列目が最終的な索引になります。
