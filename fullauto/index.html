    1. 名寄せ
      • ノンブル部分のソート・重複整理
      • 3以上連続するノンブル部分のハイフネーション
        • カンマは半角に統一されます。
    2. よみがな取得
    3. 配列読みに変換
      4. カタカナ・数字・アルファベット以外の記号は削除されます。
    4. 並べ替え
      • 最優先は配列読みの情報。
      • 配列読みが同じ場合は読みで昇順ソート。
      • 読みも同じ場合は項目の文字コード昇順（同じ文字種がまとまります）。
  • 先に子項目を作成していると正しく名寄せやソートが行われません。
